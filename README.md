# kicad plugin(s)

这里是kicad中使用的一些插件

## 1.  gen_pos_files.py  

导出器件位置信息，生成２个文件， 
文件１：所有器件。  
文件２：只有SMT器件  
本插件从https://github.com/KiCad-RU/kicad_plugins中修改而来
